\section{nbi}

Switch for activation \inputparameter{lsw\_nbi} in namelist
\inputparameter{collision}.

Other inputparameters are in own namelist \inputparameter{nbi}.
This includes \inputparameter{legmax\_nbi},
\inputparameter{name\_nbi\_data\_file} and
\inputparameter{number\_nbi\_species}.

Module \codevariable{collop\_nbi} stores the data, and namelist
parameters.

itself can not be used as there is only one velocity module dependency
(lag) for nbi part.
